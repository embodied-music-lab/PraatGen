  # Create / Read
  Create empty PointProcess: name$, start time, end time
  Create Poisson process: name$, start time, end time, density

  # Query — jitter (PointProcess only)
  Get jitter (local): from time, to time, shortest period, longest period, maximum period factor
  # Verified: Get jitter (local): 0, 0, 0.0001, 0.02, 1.3

  # Query — shimmer (requires PointProcess + Sound selected)
  Get shimmer (local): from time, to time, shortest period, longest period, maximum period factor, maximum amplitude factor
  # Verified: Get shimmer (local): 0, 0, 0.0001, 0.02, 1.3, 1.6
    Get shimmer (local_dB): from time, to time, shortest period, longest period, maximum period factor, maximum amplitude factor
  # Verified: Get shimmer (local_dB): 0, 0, 0.0001, 0.02, 1.3, 1.6
  # Provenance: Phonanium AVQI v02.06 script; Paste Commands 24 Feb 2026
  # DISAMBIGUATION NOTE:
  # To query a PointProcess point's time, use the UNIVERSAL command:
  #   Get time from index: point number
  # Do NOT use "Get time of point:" — that is a TextGrid point-tier command
  # (see COMMANDS_TextGrid.txt) and will fail on PointProcess objects.

  # Query — point-by-index navigation
  Get number of points
  Get time from index: point number
  Get low index: time
  Get high index: time
  Get nearest index: time
  # Verified: Get low index: 0.523 — returns index of last point at or before time
  # Verified: Get high index: 0.523 — returns index of first point at or after time
  # Provenance: Vibrato analysis, 18 March 2026
  # NOTE: These are PointProcess commands. The forms `Get low index from time:`
  # and `Get high index from time:` belong to TimeTier-family objects (PitchTier,
  # IntensityTier, etc.), NOT to PointProcess. Do not confuse them.

  # Modify
  Add point: time
  Add points
  Remove point: point number
  Remove point near: time
  Remove points: from point number, to point number
  Remove points between: from time, to time

  # Convert / Analyze
  To TextGrid: tier names$, point tiers$
  To PitchTier: maximum interval
  To TextGrid (vuv): maximum period, mean period
  To Sound (pulse train): sampling frequency, adaptation factor, adaptation time, interpolation depth
  To Sound (phonation): sampling frequency, adaptation factor, maximum period, open phase, collision phase, power1, power2
  To Sound (hum)
  To Matrix
  Up to TextTier: text$
  Up to PitchTier: frequency
  Up to IntensityTier: intensity
  To AmplitudeTier (point)
  To AmplitudeTier (period): from time, to time, shortest period, longest period, maximum period factor
  To H1minusH2Tier: from time, to time, shortest period, longest period, maximum period factor
  To Ltas: maximum frequency, bandwidth, shortest period, longest period, maximum period factor
  To Ltas (only harmonics): maximum harmonic, shortest period, longest period, maximum period factor

  # Draw
  Draw: from time, to time, garnish?

  # Other
  View & Edit
  View & Edit with Sound?
  Play as pulse train
  Hum
  Fill: from time, to time, period
  Voice: period, maximum voiced period
