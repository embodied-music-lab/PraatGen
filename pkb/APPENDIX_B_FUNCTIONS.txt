# RESERVED IDENTIFIERS (not functions — language constants)
# The following cannot be used as variable names, procedure parameters,
# or loop counters:
#   e         — Euler's number (2.71828...)
#   pi        — Pi (3.14159...)
#   undefined — The undefined value
# Error: "You cannot use 'e' as the name of a variable."
# Common collision: loop counters (for e from 1 to n), procedure
# parameters (.e), single-letter nested loop iterators.

#   ncol    — number of columns (Matrix, TableOfReal, Table)
#   nrow    — number of rows (Matrix, TableOfReal, Table)
#   col     — current column number in a Formula command
#   row     — current row number in a Formula command

#   row$[i] — row name (TableOfReal)
#   self    — current cell value (lowercase, no dot)
#   self$   — current cell value as string (Table)
#   Self.xmin — disambiguates from script variable of same name (capital S, dot)

# Cross-object reference syntax:
#   object[id].xmin          — attribute of object by numeric ID
#   object["Sound name"].nx  — attribute of object by type + name
